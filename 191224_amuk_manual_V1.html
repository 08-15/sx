Welcome to your new Mailbox experience

Hello; As your new assistant let me explain briefly, how we can work together smoothly and efficient. For detailed instructions, click the language flag to find out more.

 

I’m always on your service, waiting for instructions. Please note that I’ll receive your commands by return mail, means you must write the messages to your own mail account.

I listen for two kind of instructions, with X-RAY reply or new mail. X-RAY replies are the fast actions, working by returning a received X-RAY message, while for the MAIL commands you must submit a new mail. Each command is explained below in a separate step. Just study carefully, it is simple, easy to understand and to handle.

 

Important: The samples below showing “NAME” and you must replace that with my name, shown in the subject welcome message. Please take the name as shown in capital letters! Thanks.  

X-RAY command activate:

Received subject line:          X-RAY><[abc@unknown.de] NAME

Subject line to return:   RE: X-RAY>a<[abc@unknown.de] NAME
Description:
If the new message is from a sender you like to add to your community, return the X-RAY (yes, to your own email) and put an “a” between the two arrows as shown above (nothing else to change). As a result, you should get the original message now.

X-RAY command block:

Received subject line:          X-RAY><[abc@unknown.de] NAME

Subject line to return:   RE: X-RAY>b<[abc@unknown.de] NAME
Description:

If the new message is from a sender you don’t like to add to your community, return the X-RAY (yes, to your own email) and put an “b” between the two arrows as shown above (nothing else to change). As a result, you should get no further messages from that sender anymore.

MAIL command register:

NAME:register(test@abc.com, beta@aro.in, coca@cola.lola)
Description:

Register a new email address: Make a new mail and insert the requested account following above sample. You can register more than one address at a time with comma separation between the addresses. Don’t forget to change “NAME” and the address/es between the brackets!

MAIL command block:

NAME:block(test@abc.com)
Description:

Block an active sender: Make a new mail and insert the requested account following above sample.

Don’t forget to change “NAME” and the address/es between the brackets!

MAIL command activate:

NAME:activate(test@abc.com)
Description:

Activate a blocked sender: Make a new mail and insert the requested account following above sample.

Don’t forget to change “NAME” and the address/es between the brackets!

MAIL command xf off:

NAME:xf(off)
Description:

Switch xf off:  Make a new mail and insert the requested command as shown in above sample.

Don’t forget to change “NAME”!

MAIL command xf on:

NAME:xf(on)
Description:

Switch xf on:  Make a new mail and insert the requested command as shown in above sample.

Don’t forget to change “NAME”!

MAIL command print register:

NAME:print
Description:

Print a community list: If you like to check the status of all registered email accounts, send the above shown command by new mail. You should get the list as PDF in response.

Don’t forget to change “NAME”!

Brief functionality summary:

During installation I register all senders which are at that time in your inbox. I assume that this is the community you are in contact with in general. That means, those email addresses are automatically active users and I do not care about them in future at all. If there is a message from a new unregistered sender, I will inform you with a message X-RAY, what is a protected view without functionalities. Now you can make your decision, whether to activate or block the sender with the command, you'll find in the above list. If you take no any action, that specific new sender can send you two more mails, until she/he will be blocked upon the third message sent. That is the default process and we call it "xf" (X-factor). You might change it, if requested and thereafter no new sender will be blocked. For that, check for the related command above.

Thanks much for using our services. We make it simple, to protect your account!
